Objectif :

Le but de ce travail était de réaliser plusieurs visualisations à partir d’un ou plusieurs jeux de données. Nous avons décidé de réaliser des représentations de l’enneigement dans les alpes entre 1996 et 2024. Pour les deux premières visualisations, nous avons décidé d’étudier les deux mois de l’année où l’enneigement est maximal, c’est-à-dire les mois de février et de mars. Pour ce faire, nous avons regroupé tous les jeux de données de nivologie disponibles sur le site de météo France pour ces deux mois en un seul et même fichier.
